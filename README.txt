Django||Django Rest Framework
=============================

инструкция по установке локально


БИБЛЕОТЕКИ
------------
> создать виртуальное окружение и рядом расположить проект Django

> запустить команду с включенным виртуальным окружением - 'pip install -r req.txt' в каталоге с данным файлом


БАЗА ДАНЫЫХ POSTGRESQL
------------
> создать базу в postgresql

> в файле settings.py в DATABASES ввести свои параметры и сделать миграции

> запустить команду с директории где лежит файл manage.py > python manage.py runserver


АВТОРИЗАЦИЯ И РЕГИСТРАЦИЯ В DRF
------------

http://127.0.0.1:8000/auth/token/login/ - авторизация
(при передаче необходимых параметров получаем токен пользователя, в административной панели таблица Tokens)

http://127.0.0.1:8000/auth/token/logout/ - выход из аккаунта пользователя, необходимо передать токен в HEADERS в Postman Agent
(при передаче токен пользователя удаляеться)

http://127.0.0.1:8000/auth/users/ - регистрация
(нужно передать необходимые параметры и по данному email будет отправлено сообщение)
